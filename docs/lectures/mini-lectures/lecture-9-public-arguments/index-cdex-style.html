The Public Humanities Tradition

A long history of public engagement

  • Historic Roots: Public intellectuals writing for newspapers and magazines (1920s-1960s)
  • Community Archives: Local historical societies preserving neighborhood memories (1970s-1980s)
  • Oral History Projects: Recording and sharing voices from marginalized communities (1980s-1990s)
  • Digital Humanities (Now): Web-based projects democratizing access to cultural knowledge and analysis
Framework: Sacred Boundaries
Breaking down barriers between "expert" knowledge and public understanding. Accessibility as an ethical commitment.

Code as Argument

Every technical choice is a rhetorical choice

  • HTML Tags = Meaning: <h1> says "this is most important", <aside> says "this is supplementary", <strong> says "emphasize this"
  • CSS Styling = Emphasis: Larger font = more important, Bright color = demands attention, Top of page = priority information
  • Layout = Argument: Same content, three different web layouts = three different arguments about what matters most
Framework: Classification Logic
How you structure information (headers, sections, navigation) shapes how readers understand it. HTML is an act of categorization.
