Code as Argument

Every Technical Choice is a Rhetorical Choice

HTML Tags = Meaning

<h1> says "this is most important"
<aside> says "this is supplementary"
<strong> says "emphasize this"

CSS Styling = Emphasis

Larger font = more important
Bright color = demands attention
Top of page = priority information

Layout = Argument

Same content, three different web layouts = three different arguments about what matters most

Framework: Classification Logic

How you structure information (headers, sections, navigation) shapes how readers understand it. HTML is an act of categorization.
