## Pedagogical Visual Strategy

This lecture teaches **HTML fundamentals with critical DH lens**, so visuals should:
1. **Show structure as argument** - semantic tags make claims about content
2. **Reveal classification logic** - how tags categorize and hierarchize
3. **Connect technical to ethical** - accessibility code as moral choice
4. **Use architectural metaphors** - structure/blueprint/building framework
5. **Prepare for CSS lecture** - establish structure first, style next
6. **Ground in DH examples** - archives, digital projects, scholarly arguments

Color coding:
- **Purple** = HTML tags/code domain
- **Blue** = navigation/links/accessibility/ethics
- **Green** = good practices/primary/main content
- **Red** = problems/exclusion/bad practices
- **Amber** = rhetoric/emphasis/warnings/aside content
- **Gray** = neutral/footer/background

Structure-focused imagery:
- Blueprints, wireframes, architectural diagrams
- Hierarchy pyramids, organizational charts
- Classification systems, sorting mechanisms
- Pathways, navigation maps, signposts
- Before/after structural comparisons
- Annotated mockups showing rhetorical function
