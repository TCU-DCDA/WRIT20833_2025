CSS as Visual Rhetoric

How styling choices make arguments

If HTML is the argument's structure, CSS is its voice, tone, and emphasis.

From Structure to Style

HTML Provides

  • Logical structure (<header>, <main>, <aside>)
  • Content organization (headings, paragraphs, lists)
  • Semantic meaning (what things are)

CSS Provides

  • Visual hierarchy (size, color, position)
  • Emotional tone (color, typography, spacing)
  • Persuasive emphasis (what demands attention)

Together: Structure argues what matters. Style argues how much and in what way.

CSS Selectors: Choosing What to Emphasize

Selectors Choose Targets

h1 { font-size: 48px; color: #2ecc71; }

Chooses: All h1 headings

Argues: Top-level headings are the most important, should be large and green (positive/affirming)

.community-voice { font-size: 20px; font-weight: bold; }

Chooses: Elements with class "community-voice"

Argues: Community perspectives deserve visual prominence

aside { font-size: 12px; color: #95a5a6; }

Chooses: All aside elements

Argues: Supplementary content is less important (small, gray)

Critical Question

Who decides what gets styled prominently vs. minimized? What values does that encode?

Color: Emotional & Cultural Rhetoric

Color Choices Make Claims

Red (#e74c3c)

Associations: Urgency, danger, passion, importance, stop

Rhetorical use: Errors, warnings, calls to action

Cultural variation: In China, red = celebration, luck

Green (#2ecc71)

Associations: Success, growth, nature, safety, go

Rhetorical use: Confirmation, environmental topics, positive framing

Cultural variation: In Middle East, green = Islam, paradise

Blue (#3498db)

Associations: Trust, calm, professionalism, technology

Rhetorical use: Corporate sites, health, social platforms

Cultural variation: Widely positive across cultures, but depth varies

Gray (#95a5a6)

Associations: Neutral, unimportant, background, disabled

Rhetorical use: De-emphasize, minimize, push to background

Accessibility concern: Low contrast = exclusion

Takeaway: Color is never neutral. Every color choice encodes cultural values and makes emotional claims.

Typography: The Voice of Your Text

Font Choices Argue Tone & Authority

Serif Fonts (Georgia, Times)

Tone: Traditional, authoritative, academic, formal

Rhetorical use: Scholarly articles, established institutions, print tradition

Sans-Serif (Helvetica, Arial)

Tone: Modern, clean, accessible, neutral

Rhetorical use: Web interfaces, contemporary brands, digital-first

Monospace (Courier, Monaco)

Tone: Technical, code-like, precise, typewriter

Rhetorical use: Code examples, data, developer tools

Size & Weight = Importance

font-size: 48px; → Demands attention, signals priority

font-size: 12px; → Minimizes, suggests less important

font-weight: bold; → Emphasizes, argues significance

Layout & Position: Spatial Arguments

Space = Power

Center vs. Margin

text-align: center; or margin: 0 auto;

Argues: This is central, primary, for everyone

float: right; margin-left: 2rem;

Argues: This is supplementary, aside, for some

Top vs. Bottom

position: fixed; top: 0;

Argues: Always visible, always important (navigation)

position: relative; bottom: 0;

Argues: Concluding, credits, fine print (footer)

Visible vs. Hidden

display: block; or visibility: visible;

Argues: You should see this

display: none; or visibility: hidden;

Argues: This isn't for you (or isn't important enough)

Critical Question

What content gets centered? What gets pushed to margins? Who is included in visible space vs excluded through hiding?

Whitespace: The Rhetoric of Breathing Room

Space Argues About Worth

Generous Spacing

padding: 3rem; margin: 2rem 0; line-height: 1.8;

Argues: This content deserves room to breathe. It's valuable, unhurried, worthy of contemplation.

Common in: Premium brands, art galleries, thoughtful scholarship

Minimal Spacing

padding: 0.2rem; margin: 0; line-height: 1.2;

Argues: This content is utilitarian, disposable, or less important. Information density over comfort.

Common in: Classified ads, data tables, rushed communication

Accessibility Dimension

Cramped spacing creates barriers for users with cognitive differences, motor challenges, or vision impairments. Generous spacing is inclusive design.

DH Example: CSS Rhetoric in Practice

Case Study: Oral History Digital Archive

Same HTML structure, two different CSS approaches:

Approach A: Institutional Voice

CSS Choices:

  • Serif fonts (Georgia) → academic authority
  • Muted colors (#95a5a6 grays) → neutral, objective tone
  • Small testimonial text (14px) → voices as data
  • Large methodology section (24px) → scholarly process is primary
  • Generous margin around analysis → valuable scholarship
  • Cramped spacing for community quotes → less important

Argument: This is scholarly research about communities. Academic interpretation is central.

Approach B: Community-Centered Voice

CSS Choices:

  • Sans-serif fonts (Helvetica) → accessible, modern
  • Warm, vibrant colors (#e74c3c, #f1c40f) → emotional, human
  • Large testimonial text (20px) → voices are primary
  • Small methodology note (12px) → process is secondary
  • Generous spacing around testimonials → valuable stories
  • Compact academic framing → context, not focus

Argument: This is community storytelling with scholarly support. Lived experience is central.

Lesson: Identical content + identical structure + different CSS = completely different rhetorical stance and argument about whose voice matters.

Responsive Design: Who Gets Access?

CSS Media Queries = Access Decisions

Desktop-Only Design (No Responsive CSS)

No @media queries, fixed widths, assumes large screens

Excludes: Mobile users, tablet users, users with small screens

Argues: Desktop users are the intended audience. Others are not important.

Mobile-First Responsive Design

Base styles for mobile, @media (min-width: 768px) for larger screens

Includes: All users, prioritizes mobile (majority of web traffic globally)

Argues: Access is a right, not a privilege. We design for everyone.

Global Digital Divide

In many countries, mobile is the only internet access. Desktop-only design is a form of global exclusion.

Ethical Stance: Responsive CSS isn't optional. It's a commitment to access justice.

CSS as Rhetorical Practice

What You Now Understand

  • CSS selectors choose what to emphasize (targeting = rhetorical choice)
  • Color encodes emotion and cultural meaning (never neutral)
  • Typography gives text a voice and tone (font = personality)
  • Layout positions content in power hierarchies (space = value)
  • Whitespace argues about worth (breathing room = importance)
  • Responsive design determines access (inclusion vs exclusion)
  • All styling choices are arguments about what matters and to whom

Ask Yourself While Styling

  • What am I making large/bold/colorful? What does that center?
  • What am I making small/gray/hidden? Who does that marginalize?
  • Whose cultural associations do my color choices assume?
  • Does my design work on mobile devices? Who am I excluding if not?
  • Does my spacing create accessibility or barriers?
  • What argument am I making about whose voice deserves emphasis?
