## General Style Guidelines

**Color Palette (consistent across all images):**
- Primary Purple: #9b59b6 (CSS code/selectors)
- Light Purple: #c9a9ff (syntax/code elements)
- Blue: #3498db (accessibility/ethics/trust)
- Green: #2ecc71 (good practices/inclusion/success)
- Red: #e74c3c or #ff6b6b (problems/exclusion/urgency)
- Yellow/Amber: #f1c40f (rhetoric/critical questions/warnings)
- Gray: #95a5a6 (neutral/minimal/marginalized)
- Dark Background: #1a0330 to #2d1b5e (when needed)

**Aesthetic Principles:**
- Show CSS as argumentative styling, not neutral decoration
- Color, typography, layout = rhetorical choices
- Visual emphasis = claims about importance
- Spacing = value judgments
- Responsive design = access ethics
- Every style choice encodes cultural meaning
- Design with critical awareness
- Connects visual rhetoric to DH practice

**Thematic Emphasis:**
- CSS selectors choose what to emphasize (targeting = power)
- Color encodes emotion and cultural meaning (never neutral)
- Typography gives content voice and personality
- Layout creates spatial hierarchies (center vs margin)
- Whitespace argues about worth (generous vs cramped)
- Responsive design determines access (inclusion vs exclusion)
- All styling choices are arguments about value
- Critical design practice integrates ethics and aesthetics

## Pedagogical Visual Strategy

This lecture teaches **CSS fundamentals with rhetorical and ethical lens**, so visuals should:
1. **Show style as argument** - visual choices make claims about importance
2. **Reveal emotional/cultural coding** - color, font, spacing carry meaning
3. **Connect aesthetics to ethics** - accessibility, inclusion, cultural sensitivity
4. **Use before/after comparisons** - same content, different styling, different arguments
5. **Ground in DH examples** - oral histories, archives, digital scholarship
6. **Prepare for integration** - structure (HTML) + style (CSS) = complete rhetorical practice

Color coding:
- **Purple** = CSS code/syntax
- **Amber** = rhetoric/critical questions/emphasis
- **Blue** = accessibility/ethics/trust
- **Green** = good practices/inclusion/success
- **Red** = problems/exclusion/urgency
- **Gray** = neutral/minimal/marginalized
