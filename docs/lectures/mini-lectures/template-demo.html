Image-Left Layout Example

Content Goes Here

This layout places the image on the left (40% width) and text content on the right (60% width). The image scales proportionally to fit its column.

  • Image auto-scales to available space
  • Max height: 550px on desktop
  • Stacks vertically on mobile devices
  • Text column remains readable and well-spaced

Key Insight: Use this layout when the image should introduce or set context for the text content

Image-Right Layout Example

Text Content First

This layout reverses the order: text content on the left (60% width) and image on the right (40% width). Better for text-driven explanations.

Multi-Slide Content Pattern

First Set of Concepts

When you have complex topics requiring more than 5-6 bullet points, split the content across multiple slides at natural concept boundaries.

  • This slide demonstrates the "Part 1" pattern
  • Notice the continuation badge at the top
  • Each part should feel complete but connected
  • Use text-section components for organization

Second Set of Concepts

You can include multiple text-section components on a single slide to organize related ideas.

  • Each section has its own visual container
  • Left border accent maintains consistency
  • Headings use theme colors (#c9a9ff)
