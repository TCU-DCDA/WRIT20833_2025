#### **CodeAlong 2: WRIT20833_HTML_01_First_Website_F25.ipynb**
**Learning Goals**: Create first HTML page for text analysis showcase
- **Technical Skills**:
  - Basic HTML structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
  - Essential semantic tags: `<h1>-<h6>`, `<p>`, `<div>`, `<section>`
  - Document metadata: `<title>`, `<meta>` descriptions
- **Humanities Integration**:
  - Create landing page introducing their cultural dataset
  - Present HW4-1 term frequency results in structured HTML
  - Academic context: "Digital Portfolio Introduction"
- **Deliverable**: Simple HTML page showcasing top 10 most frequent words with context

#### **CodeAlong 3: WRIT20833_HTML_02_Content_Structure_F25.ipynb**
**Learning Goals**: Organize cultural analysis content for web presentation
- **Technical Skills**:
  - Lists (`<ul>`, `<ol>`, `<li>`) for structured data presentation
  - Links (`<a>`) to sources, references, and methodology
  - Images (`<img>`) for visualizations created in previous assignments
  - Tables (`<table>`) for presenting analytical results
- **Humanities Integration**:
  - Structure sentiment analysis findings from HW4-1
  - Create navigation between different analytical sections
  - Present methodology and critical reflection
- **Deliverable**: Multi-page site structure with term frequency, sentiment analysis, and methodology sections
