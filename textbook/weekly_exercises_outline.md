# Weekly Exercises Outline Template

## Exercise Design Principles
- **Scaffolded Learning**: Each exercise builds on previous weeks
- **Real-World Context**: Use contemporary, relevant datasets
- **Immediate Feedback**: Results should be visible and interpretable
- **Incremental Complexity**: Start simple, add layers each week
- **Collaborative Opportunities**: Mix individual and group activities

## Weekly Exercise Structure Template

### Pre-Class Preparation
- [ ] Reading assignment (textbook chapter)
- [ ] Dataset exploration (basic familiarity)
- [ ] Setup check (environment, files)

### In-Class Activities (3 hours total)
- **Hour 1**: Concept introduction and guided practice
- **Hour 2**: Hands-on coding with instructor support
- **Hour 3**: Application exercise and sharing/discussion

### Post-Class Assignment
- [ ] Extension of in-class work
- [ ] Reflection questions
- [ ] Preparation for next week

## Week-by-Week Exercise Outlines

### Week 1: Environment Setup & Variables
**Learning Goals**: Comfortable with Colab, understand variables and basic data types

**Exercise 1** (30 min): "Your Digital Identity"
- Create variables for name, favorite song, birth year
- Calculate age, format personal introduction
- Print formatted output

**Exercise 2** (45 min): "Pop Culture Math"
- Calculate streaming statistics (hours listened, cost per song)
- Work with strings (artist names, song titles)
- Basic error troubleshooting

**Exercise 3** (60 min): "First Text Analysis"
- Count characters in favorite quotes
- Simple string manipulation (uppercase, lowercase)
- Introduction to print formatting

**Takeaway**: Students leave with working Colab notebook and confidence in basic operations

### Week 2: Lists and Loops
**Learning Goals**: Understand lists, iterate through data, basic text processing

**Exercise 1** (30 min): "Playlist Analysis"
- Create list of favorite songs
- Loop through and print each song
- Count total songs, find longest title

**Exercise 2** (45 min): "Social Media Posts"
- Process list of sample tweets/posts
- Count words in each post
- Find posts above/below character limits

**Exercise 3** (60 min): "Trend Tracking"
- Loop through list of hashtags
- Count occurrences of specific terms
- Create simple frequency report

**Takeaway**: Comfort with lists and loops, ready for real data

**Implementation Note (September 2025)**: Existing Tutorial_02_Classification_Logic notebook demonstrates effective integration of technical concepts (conditionals, boolean logic) with critical analysis of how code classifies people

### Week 3: Functions and File Handling
**Learning Goals**: Write reusable functions, read external data files

**Exercise 1** (30 min): "Text Processing Functions"
- Write function to clean text (remove punctuation)
- Function to count words
- Function to find longest word

**Exercise 2** (45 min): "Reading Real Data"
- Load CSV file of song lyrics
- Apply cleaning functions to data
- Basic data exploration (how many songs, average length)

**Exercise 3** (60 min): "Custom Analysis Pipeline"
- Combine functions into analysis workflow
- Process multiple files
- Generate summary report

**Takeaway**: Understanding of functions and file I/O, ready for pandas

**Implementation Note (September 2025)**: Existing Variables_DataTypes and StrMethods_Conditionals_Loops notebooks provide comprehensive foundation for this week's content

### Week 4: Dictionaries and Data Organization
**Learning Goals**: Master dictionaries, organize complex data, prepare for pandas

**Exercise 1** (30 min): "Artist Discography Dictionary"
- Create nested dictionaries for artists and albums
- Practice key-value access and modification
- Count songs by artist using dictionary methods

**Exercise 2** (45 min): "Social Media Profile Data"
- Build dictionary representing user profiles
- Process multiple profiles using loops
- Find patterns in profile data (most common interests, etc.)

**Exercise 3** (60 min): "Cultural Event Database"
- Create complex nested structure for events
- Search and filter using dictionary operations
- Export data to prepare for Week 5 pandas introduction

**Takeaway**: Strong foundation in dictionaries, ready for structured data analysis

### Week 5: Libraries and Visualization
**Learning Goals**: Use pandas and matplotlib, create first visualizations

**Exercise 1** (30 min): "DataFrame Basics"
- Load movie dataset with pandas (building on Week 4 dictionary concepts)
- Explore structure (head, info, describe)
- Basic filtering and sorting

**Exercise 2** (45 min): "First Charts"
- Create bar chart of movie genres
- Line plot of release years
- Customize colors and labels

**Exercise 3** (60 min): "Data Story"
- Combine multiple visualizations
- Add titles and annotations
- Export charts for sharing

**Takeaway**: Comfort with pandas and basic plotting, ready for NLP

### Week 6: Advanced Text Analysis
**Learning Goals**: Sentiment analysis, text preprocessing, introduction to statistics

**Exercise 1** (30 min): "Sentiment in Song Lyrics"
- Apply sentiment analysis to music data
- Compare sentiment across genres/decades
- Visualize sentiment trends

**Exercise 2** (45 min): "Text Preprocessing Pipeline"
- Clean text data (punctuation, stopwords)
- Normalize text for analysis
- Compare before/after processing

**Exercise 3** (60 min): "Statistical Foundations"
- Basic descriptive statistics for text data
- Understanding distributions and outliers
- Error handling for missing/bad data

**Exercise 2** (45 min): "Social Media API Practice"
- Access Twitter/Instagram public APIs

**Exercise 3** (60 min): "Building Your Own Dataset"

## Assessment Criteria for Exercises

### Technical Skills (40%)
- Code runs without errors
- Appropriate use of programming concepts
- Code is readable and well-commented

### Analysis & Interpretation (40%)
- Meaningful insights from data
- Clear explanation of findings
- Appropriate choice of visualization/analysis method

### Communication (20%)
- Clear writing in comments and explanations
- Effective presentation of results
- Thoughtful reflection on process

## Exercise Development Checklist

For each week:
- [ ] Learning objectives clearly defined
- [ ] Datasets identified and prepared
- [ ] Exercise instructions written (step-by-step)
- [ ] Solution code developed and tested
- [ ] Common error scenarios anticipated
- [ ] Extension activities for fast finishers
- [ ] Assessment rubric created
- [ ] Connection to previous/next weeks explicit

## Collaborative Exercise Ideas

### Peer Learning Opportunities
- **Code Review Partners**: Students review each other's solutions
- **Data Detective Groups**: Different groups analyze same dataset, compare findings
- **Teaching Moments**: Students explain concepts to classmates
- **Gallery Walks**: Students present visualizations, give feedback

### Class-Wide Projects
- **Collective Dataset**: Each student contributes data for shared analysis
- **Comparison Studies**: Different groups analyze different platforms/time periods
- **Community Resource**: Final projects form coherent resource for public use
